# IMDB Sentiment Analysis using NLP

This project performs sentiment analysis on IMDB movie reviews.

## Tools Used
Python
Pandas
Scikit-learn
TF-IDF
Logistic Regression

## Steps
1. Load IMDB dataset
2. Convert ratings into sentiment labels
3. Apply TF-IDF vectorization
4. Train Logistic Regression model
5. Predict review sentiment

## Example Prediction
Input: "This movie is amazing"
Output: Positive
